Drive & Ride Rentals 🚗🏍️

A sleek, responsive, and fully interactive vehicle rental landing platform built as a web development mini-project. 

🌟 Key Features
* **Dynamic Showroom Navigation:** Implemented a multi-level Bootstrap Collapse architecture allowing users to click broad vehicle segments (Sedans, SUVs) to seamlessly reveal distinct model availability (Swift Dzire, BMW, Thar, etc.).
* **Dynamic Validation Modal:** Integrated fully interactive JavaScript listeners to pass individual vehicle specifications and rental rates into a unified transaction terms modal, handling user input validation dynamically.
* **Modern UI Elements:** Complete navigation bar customized with high-contrast, pill-shaped action triggers and animated user hover scaling effects.
* **Infinite Hero Loop:** Fluidly integrated a fully configured Bootstrap Carousel component looping core value propositions and fleet highlights.

🛠️ Tech Stack & Concepts Applied
* **Frontend Layout:** HTML5, Custom CSS3 Grid & Flexbox alignment
* **UI Framework:** Bootstrap v5.3 (Responsive break-point configurations, components, and utility classes)
* **Interactivity:** Native JavaScript (DOM Manipulation, conditional state handling, event listeners)

 📁 Project Structure Overview
* `index.html` - Core structural markup and layout engine
* `style.css` - Custom styling extensions, color overrides, and animations
* `images/` - Directory holding optimized assets for fleet categories and banners
